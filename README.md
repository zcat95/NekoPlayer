# Neko Player

一款基于 **Jetpack Compose** 与 **Miuix** 组件库构建的极简本地短视频播放器。

---

## ✨ 核心特性

* **智能渲染**：全面采用 `TextureView` 架构，支持高性能 UI 滤镜捕获，消除滑动闪烁。
* **交互美学**：深度整合 **Miuix** 组件库，带来原生级的卡片式交互与视觉对齐。
* **深度自定义**：支持模糊强度、遮罩高度及顶部 UI 边距的精细调节。
* **沉浸体验**：具备全屏裁切、自动静音、倍速切换等多种播放逻辑。

## 🛠️ 技术栈

* **播放引擎**：Media3 ExoPlayer
* **UI 框架**：Jetpack Compose
* **组件规范**：[Miuix](https://github.com/compose-miuix-ui/miuix)
* **核心滤镜**：`miuix-blur` (毛玻璃效果)

---

## 📜 版本历史

### v2.0.0
* **视觉重构**：集成 `miuix-blur`，支持模糊强度与高度动态调节。
* **引擎升级**：切换至 `TextureView` 渲染，解决 Compose 兼容性导致的画面缺失。
* **布局自由**：新增标签水平/垂直边距调节，支持 UI 位置自定义。
* **交互优化**：设置面板限制为 75% 屏幕高度，支持边调边看的即时预览。
* **规范化**：按照 Miuix 标准重构设置页卡片布局。

### v1.5.0
* **设置面板**：新增中心化偏好管理入口。
* **播放控制**：支持随机播放、自动续播及单视频循环。
* **功能扩展**：新增后台音频播放、启动自动静音、画面比例切换 (Crop/Fit)。
* **倍速逻辑**：支持全局 2.0x 与长按倍速模式切换。
* **数据中心**：新增本地视频总量统计与收藏计数。
* **体验优化**：重构手势逻辑，解决滑动切换与长按倍速的交互冲突。
