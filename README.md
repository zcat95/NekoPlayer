# Neko Player 

一款极致简单、小巧的本地短视频播放器。采用类似短视频 App 的交互逻辑，告别繁杂控件，为你提供最纯粹的沉浸式本地视频播放体验。

## ✨ 核心特性

* **沉浸式全屏**：无状态栏、无底部导航、无标题、无进度条。只有轻量级顶部遮罩用于状态指示。
* **丝滑手势控制**：
    * 上下滑动：无缝切换上一个/下一个视频。
    * 左右滑动：在“全部视频”和“已收藏视频”列表间快速切换。
    * 单击：暂停 / 播放。
    * 长按：2.0x 倍速播放（松手即刻恢复 1.0x）。
* **智能断点记忆**：自动保存选择的本地目录，每次启动精准恢复至上次播放的最后一个视频。
* **轻量级点赞库**：右下角独立收藏按钮，一键将视频归档至“已收藏视频”列表。
* **高自由度设置面板**：
    * **画面比例**：全屏裁切 (Crop) / 原比例完整显示 (Fit)。
    * **播放逻辑**：随机播放、自动续播、单视频无限循环。
    * **倍速偏好**：长按 2 倍速 / 全局强制 2 倍速。
    * **后台与音量**：支持切后台继续播放；支持启动默认静音（按物理音量键即刻解除）。
    * **数据统计**：直观展示已扫描及已点赞的视频总数。

## 🛠️ 技术栈

* **语言**: Kotlin
* **UI 框架**: Jetpack Compose
* **播放器内核**: AndroidX Media3 (ExoPlayer)
* **架构设计**: 极简组件化，严格的生命周期与内存管理 (滑动即时销毁播放器实例)

## 🚀 编译与运行

1. 克隆本仓库到本地。
2. 使用 Android Studio 打开项目。
3. 等待 Gradle 同步完成。
4. 点击 `Run` 编译并安装到 Android 设备（要求 Minimum SDK API 24+）。
5. 首次进入点击屏幕中央按钮授予目录访问权限，即可开始体验。
